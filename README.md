# Cristella's Personal Portfolio (Responsive Design)

## Overview

In this assignment, I enhanced my personal portfolio website by applying advanced CSS techniques in order to ensure full responsiveness and adaptability. I added media queries, flexbox/grid techniques and other elements in order to create a fully responsive website on both mobile devices and desktop. I attempted to enhance the user experience by building on the previous website created.

The website reflects my personal interest in fashion while maintaining a clean, readable, and structured design.

## Responsive Design Approach

I followed a **mobile-first design strategy**, which meant the base of the website was made for smaller screens, then took larger ones into consideration by applying different elements and ensuring responsiveness on both devices. Editing and restructuring text and fonts also contributed to the adaptable layout.

### Breakpoints used:
- **600px (Tablet):**
  - Navigation layout changes from vertical to horizontal using Flexbox
  - Font sizes adjusted for better readability

- **900px (Desktop):**
  - Main content is centered and expanded using max-width
  - Layout becomes more structured and visually balanced

## Layout Techniques (Flexbox & Grid)

### Flexbox
Flexbox was used in key areas of the website:
- **The navigation bar:**
  - The bar sed `display: flex` to align all navigation links
  - Adjusted layout from column (mobile) to row (larger screens)
- **Image sections:**
  - Images are arranged using Flexbox to stack on smaller screens while aligning horizontally on larger screens
  - 

## Navigation & Interactivity

The custom navigation menu resembles buttons
- Background colours, margin and padding were used to create a button-like appearance
- `border-radius` was applied for a more modern look (rounding out the edges)
- `display: inline-block` ensured proper spacing and alignment preventing the overlapping or cluttered look

### Transitions:
- A hover effect was added to navigation links using:
  - `transition`
  - `transform: scale()`
- This resulted in the buttons changing color while also hovering when a user's cursor made contact with it.

## List Styling

A **Resources section** was added using an unordered list (`ul`), and styled to improve visual appeal and add diversity to the website.
- Default bullet points were removed
- Each list item (`li`) was styled as a separate block to enhance the attractiveness of the website using:
  - `background-color`
  - `padding`
  - `margin`
  - `border-radius`
- This created a clean, card-like layout.

## Design Choices

### Colour Scheme
- The soft pink background reflects a modern, creative aesthetic while matching my theme of my website 
- Light green navigation elements provided contrast and visual interest to appeal users

### Typography
- Headings use a custom Google Font for personality and uniqueness while improving readability
- Paragraph text uses a simple font (Arial) to maintain readability 

### Layout & Spacing
- Margin and padding were used to create space and prevent clutter ensuring good readability 
- Content is centered and constrained on larger screens for a more professional layout ensuring responsiveness
